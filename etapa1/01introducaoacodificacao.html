Introdução a Codificação

Desenvolvedor de Software

O desenvolvedor de software é o profissional encarregado da codificação no desenvolvimento do software. Eles identificam as soluções de software escrevendo o código necessário para rodar a aplicação ou sistema.

Desenvolvedores são proficientes em:

  • Codificação
  • Teste
  • Diferentes linguagens de programação
  • Várias plataformas de desenvolvimento
  • Sabem como estruturar código

Em essência, o desenvolvedor de software é tido como o indivíduo que junta as funcionalidades (requisitos) e escreve o código necessário para criar a aplicação de software ou o Sistema.

Iremos aqui implmentar 2 projetos: um projeto de um website e um aplicativo web. Para aqueles que são inciantes acho importante começar com o projeto de um website, mas para aqueles que já tem conhecimento na área de desenvolvimento sugiro seguir para o desenvolvimento do aplicativo.

Projeto de desenvolvimento de um website: Portifólio

Neste projeto passaremos por 5 grandes temas:

  1. Fases relacionadas com o projeto de um site
  2. Fases relacionadas com ambiente de desenvolvimento
  3. Fases relacionadas com o desenvolvimento do site no Front-End:
    • A Linguagem de marcação HTML
    • A Linguagem de estilização CSS
    • Framework Boostrap
    • A Linguagem de programação Javascript
  4. Fases relacionadas com o desenvolvimento do site no Back-End:
    • A Linguagem de programação PHP
    • O Banco de dados Mysql
  5. Fases relacionadas com a publicação do site

O objetivo aqui e tornar você um Desenvolvedor Full Stack

O que é um desenvolvedor full stack? O desenvolvedor full stack é capaz de trabalhar com toda a pilha de desenvolvimento de um projeto. Além de conseguir atuar com o font-end e o back-end, esse profissional também pode contribuir em todas as etapas e partes de um sistema, como o servidor e o banco de dados.
